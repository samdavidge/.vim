My attempt at migrating to vim, wish me luck.

## Installation
```
$ cd ~
$ git clone ://github.com/samdavidge/.vim.git
$ rm -rf ~/.config/nvim
$ ln -s ~/.vim ~/.config/nvim
$ nvim
```

Be sure to run `:checkhealth` to identify any dependency issues (node, python, php, etc)
